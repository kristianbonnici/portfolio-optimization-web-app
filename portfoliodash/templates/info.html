App Info

What is this web app about?

This is a web app for generating optimal portfolios based on Sharpe Ratio. The ratio is a metric that compares the amount of return versus the amount of risk, based on historical data.

What is a good Sharpe Ratio?

Sharpe Ratio above 1.00 is commonly considered to be "good". This suggests that the portfolio offers excess returns relative to its risk.

Measurement details

\( \large{ \text{ Sharpe Ratio } = {R_p - R_f \over \sigma} } \)

\( \text{ Where: } \)

  • \( R_p = \text{ annualized return of daily returns } \)
  • \( R_f = \text{ annual risk-free return } \)
  • \( \sigma = \text{ annualized standard deviation of daily returns } \)

Limitations

  • Standard deviation as a proxy of total portfolio risk assumes that the daily returns are normally distributed. However, this is not always the case.
  • The Sharpe Ratio can be manipulated by lengthening the measurement interval. For example, using annualized standard deviation of weekly or monthly returns will generally result in lower volatility than with daily returns.
  • It's possible to cherry-pick a period for the analysis with the best potential Sharpe Ratio, rather than a neutral look-back period.
